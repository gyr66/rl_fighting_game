# rl_fighting_game

国科大强化学习格斗游戏大作业

## Setup

Ctrl + Shift + P => Reopen in Container

```sh
pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple"
```

## Run

## Train PPO

Change the arguments in train_ppo.sh

```sh
bash train_ppo.sh
```

## Train MCTS

Modify Config.py

```sh
bash train_mcts.sh
```

## Eval

Change the arguments in eval.sh

```sh
bash eval.sh
```
